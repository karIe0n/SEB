Exchange Rates HTML Viewer

This Python script fetches exchange rates from the European Central Bank (ECB), calculates average historical rates, generates an HTML table, and opens it in your default web browser.


Features

    Fetch daily exchange rates from ECB.

    Fetch historical exchange rates from ECB.

    Calculate average rates for selected currencies.

    Generate a HTML table displaying today's rate and average rate.

    Open the table automatically in the default web browser.


Requirements

    Python 3.7+

    pandas library


Install dependencies via pip:

    pip install pandas


Usage

    1. Save the script as exchange_rates.py

    2. Run the script:

    python exchange_rates.py
